
# ClickOutside

React click outside component.

## Installation

```
$ npm install tj/react-click-outside
```

## Example

```js
<ClickOutside onClickOutside={::this.close}>
  <p>Im a menu or something that you want to hide when clicking outside.</p>
</ClickOutside>
```
